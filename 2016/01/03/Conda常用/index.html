Conda常用

发表于 2016-01-03   |   分类于 编辑器   |  

Anaconda是一个用于科学计算的Python发行版，支持 Linux, Mac, Windows系统，提供了包管理与环境管理的功能，可以很方便地解决多版本python并存、切换以及各种第三方包安装问题。

conda加速

基本命令

  • conda -V：显示版本
  • conda list：显示安装包
  • conda install [包名]：安装包
  • conda update [包名]：升级包

虚拟环境

  • conda create -n 环境名 python=版本号：创建
  • activate 环境名：激活环境
  • deactivate：关闭环境
  • conda rwmove -n 环境名 -all：删除环境

基本包

  • autopep8
  • isort
  • ipython
